# 🔥 CodeBlaze — AI Adaptive Learning Path Generator

A fully working, production-ready adaptive learning platform powered by **Claude AI**, **SQLite**, **Express**, and **React**.

---

## ✅ What's Built

| Layer | Technology | Status |
|---|---|---|
| Frontend | React 19 + Vite + TailwindCSS + D3.js | ✅ Complete |

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Configure environment
Edit `.env` and set your Anthropic API key:
```
Grok_API_KEY=your_grok_api_key_here
JWT_SECRET=any-random-secret-string
PORT=3000
```
Get a FREE API key at: https://console.groq.com/keys

### 3. Run the app
```bash
npm run dev
```

Open http://localhost:3000 — the database is auto-created on first run.

### 4. Production build
```bash
npm run build
npm start
```

## 🧠 Features

### Adaptive Learning (IRT-inspired)
- Quiz difficulty auto-adjusts based on current mastery score
- Lower mastery → easier questions; higher mastery → harder questions
- IRT weighted-average mastery updates: newer attempts have higher weight

### AI Question Generation (Claude)
- Topic-specific MCQ questions generated by `gemini-1.5-flash` (free tier)
- Questions cached in SQLite to reduce API calls on repeat quizzes
- Fallback questions available when API is unavailable

### Knowledge Graph (D3.js)
- Interactive force-directed graph of all topics in your domain
- Nodes colored by mastery: green (≥80%), blue (≥60%), amber (≥40%), grey (0%)
- Click any node to jump directly to that topic's quiz
- Shows prerequisite relationships as directed edges

### Practice Lab
- AI generates a personalized coding exercise based on your mastery level
- Built-in JavaScript code runner (sandboxed `eval` with console capture)
- Progressive hints system (reveal one at a time)
- Full solution reveal available

### Analytics Dashboard
- 30-day quiz score history chart
- Topic mastery heatmap (all topics in your domain)
- Weak areas detection (mastery < 60%)
- Predicted completion date based on your learning velocity
- Learning streak tracker

## 🗄️ Database

The SQLite database is automatically created at `data/codeblaze.db` on first run.
It seeds **35 topics** across 5 domains:
- **DSA**: Arrays, Linked Lists, Trees, Graphs, DP, etc.
- **ML**: Regression, Neural Networks, NLP, CNN, etc.
- **WebDev**: HTML/CSS, JavaScript, React, Node.js, Security
- **SystemDesign**: CAP theorem, Load Balancing, Caching, Microservices
- **CloudComputing**: Docker, Kubernetes, Serverless, Networking

## 🔌 API Reference

```
POST   /api/auth/register       Register new user
POST   /api/auth/login          Login
GET    /api/auth/me             Get profile (auth required)
PUT    /api/auth/profile        Update profile / complete onboarding

GET    /api/learning/domains    List all available domains
GET    /api/learning/topics     Topics for user's domain (with mastery)
GET    /api/learning/knowledge-graph   D3-ready graph data
GET    /api/learning/learning-path     AI-generated recommendations
GET    /api/learning/practice/:id      AI coding exercise for topic
GET    /api/learning/analytics         Full analytics for dashboard

GET    /api/quiz/diagnostic     Generate diagnostic assessment
GET    /api/quiz/topic/:id      Generate adaptive topic quiz
POST   /api/quiz/submit         Submit answers, update mastery
GET    /api/quiz/history        Past quiz attempts

GET    /api/health              Server health check
```
